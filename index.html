Open a text editor (Notepad on Windows, TextEdit on Mac, or any code editor) 3. Paste the code 4. Save as: `index.html` (Make sure to save with `.html` extension, not `.txt`) ### Step 2: Deploy to Netlify (FREE) 1. Go to https://www.netlify.com 2. Sign up with email or Google 3. Click "Sites" → Drag and drop your `index.html` file 4. Get your live URL instantly! (e.g., `yoursite.netlify.app`) ### Step 3: Update Razorpay Key (Before Accepting Real Payments) 1. Sign up at https://razorpay.com 2. Complete KYC verification 3. Go to Settings → API Keys 4. Copy your Test Key ID 5. In the HTML file, find line with: `key: "rzp_test_YOUR_KEY_ID"` 6. Replace `YOUR_KEY_ID` with your actual key 7. Re-upload to Netlify ### Step 4: Customize Products Edit the `products` array in the JavaScript section to add your own jewelry items with: - Product names - Prices - Descriptions - Categories - Your own product image URLs --- ## What's Included ✅ Full e-commerce functionality ✅ Shopping cart system ✅ Product filters (category, material, price) ✅ Razorpay payment integration ✅ Mobile responsive design ✅ Dark mode support ✅ Professional UI with Duhitri branding Your Duhitri Jewellery store is ready to go live! 🚀
